# Atcoder

## Usage: Atcoder

### Start new contest

```
$ cargo compete new abc275
$ cd abc275
```

### submit

```
$ cargo compete submit a
```

## Usage: Other Contest

example problem: abc135_e

### Start new problem

```
$ ./new_problem.sh abc135_e
```

### Edit and test

```
$ cd abc135_e
# edit src/main.rs
# edit runtest(test script)
$ ./runtest
```

### Finish when passed

```
$ ./pass_problem.sh
```
